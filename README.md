🧑‍💻 **Portfolio Website (Vercel Deployment, Custom GoDaddy Domain)**  
📌 Preview  
A personal developer portfolio hosted on Vercel and connected to a custom GoDaddy domain, designed to showcase my projects, skills, and experience through a clean, responsive, and modern interface.

📖 About the Project  
This portfolio serves as my central online presence, highlighting my work, technical background, and projects in a simple and professional layout.

A key part of this project was working with an existing codebase rather than starting from scratch, which required understanding its structure and making safe, targeted improvements without breaking functionality.

🚀 What I Learned  
- Working with and modifying existing codebases  
- Understanding unfamiliar project structure and flow  
- Making safe, incremental changes in a production-style setup  
- Deploying and managing projects on Vercel  
- Connecting and configuring a custom GoDaddy domain  
- Improving UI/UX and responsiveness for a production portfolio  
- Structuring content for clear technical presentation  

📍 The Process  
I first explored the existing codebase to understand its structure and how the project was built. After getting familiar with it, I made small, careful improvements to layout, content, and UI.

Once the updates were complete, I deployed the site on Vercel and connected a custom GoDaddy domain by configuring DNS settings. Final refinements focused on responsiveness, consistency, and overall polish.

🛠️ Technologies Used  
- Next.js / React  
- Vercel  
- GoDaddy Domain & DNS  
- HTML5  
- CSS3 / Tailwind CSS
